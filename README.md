# CerealKillers

This website is for filing incident initial and incident follow-up reports pertaining to various types of cereals in line with a serial killer theme.

This application is built upon Rails 6.

## Technologies

* Rails 6

* Ruby 2.6.5

* PostgreSQL

* AWS S3

* Bootstrap 4.5

## Installation

`bundle install`

`rake db:create`

`rake db:migrate`

`rails s`
